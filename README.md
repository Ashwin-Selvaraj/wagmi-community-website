# 🚀 WAGMI Community Website

Official website for the **WAGMI Web3 community** — a grassroots Ethereum initiative based in **Coimbatore & Tiruppur**.

---

## 🌐 About

WAGMI is a community-driven initiative focused on **onboarding beginners into Ethereum and Web3**, while also supporting developers through:

- Workshops  
- Meetups  
- Hands-on learning sessions  

Our goal is to build a **strong local ecosystem** where individuals can learn, connect, and grow within the decentralized world.

---

## ✨ Features

- 🧠 Community overview and mission  
- 📅 Past and upcoming events  
- 🛠️ Developer onboarding initiatives  
- 🗺️ Roadmap and ecosystem vision  
- 🤝 Community engagement links  

---

## 🛠️ Tech Stack

- **Next.js** – React framework for scalable frontend  
- **Tailwind CSS** – Utility-first styling  
- **Framer Motion** – Smooth animations and interactions  

---

## 🎯 Purpose

To build a **sustainable Ethereum ecosystem** at a grassroots level by:

- Making Web3 accessible to beginners  
- Encouraging developer participation  
- Creating consistent community engagement  

---

## 🌍 Vision

From curiosity to contribution — helping individuals enter Web3 and grow into active builders within the global Ethereum ecosystem.
